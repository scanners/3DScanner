Project Description

We will implement a three-dimensional scanner that will generate a point cloud using triangulation. These data points will be used by a software application, which we will also implement, to transform the data points into a polygonal mesh using mathematical formulas. This polygonal mesh will represent the scanned object and be available for use in 3-D modeling software applications.

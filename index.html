Project Description

We implemented a 3D scanner using a webcam, motor, and laser line. The scanner scans an object with the laser line and collects image data through the webcam. This data is processed using mathematical calculations and generates a file defining the 3D point cloud. This point cloud can then be displayed onscreen and manipulated to create a 3D mesh.
